## 📖 Miniguia de Estudo

### 🧠 Resumo

Educação financeira é a capacidade de entender e gerenciar o dinheiro de forma inteligente. Isso envolve controlar receitas e despesas, planejar o orçamento, evitar dívidas e investir com objetivos claros.

---

### 📘 Glossário

- Receita: dinheiro recebido
- Despesa: dinheiro gasto
- Orçamento: planejamento financeiro
- Reserva de emergência: dinheiro guardado para imprevistos
- Investimento: aplicação de dinheiro para gerar retorno
